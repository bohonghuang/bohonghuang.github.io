常用条件语句

条件语句 含义
[[ -z STRING ]] 字符串是否为空
[[ -n STRING ]] 字符串是否非空
[[ STRING == STRING ]] 字符串是否相等
[[ STRING != STRING ]] 字符串是否不相等
[[ NUM -eq NUM ]] 数值相等
[[ NUM -ne NUM ]] 数值不相等
[[ NUM -lt NUM ]] 数值小于
[[ NUM -le NUM ]] 数值小于等于
[[ NUM -gt NUM ]] 数值大于
[[ NUM -ge NUM ]] 数值大于等于
[[ STRING = ~ STRING ]] 字符串是否匹配正则表达式

文件条件语句

条件 含义
[[ -e FILE ]] 存在
[[ -r FILE ]] 可读
[[ -h FILE ]] 是否为符号链接
[[ -d FILE ]] 是否为文件夹
[[ -w FILE ]] 可写
[[ -s FILE ]] 文件是否大于0字节
[[ -f FILE ]] 是否为文件
[[ -x FILE ]] 是否为可执行文件
[[ FILE1 -nt FILE2 ]] 文件1是否比文件2更晚访问
[[ FILE1 -ot FILE2 ]] 文件2是否比文件1更晚访问
[[ FILE1 -ef FILE2 ]] 是否为相同文件
